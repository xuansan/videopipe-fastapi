## 🔄 前后端交互流程

### 1️⃣ 用户认证流程
```
用户浏览器
    │
    │ 1. POST /dev-api/login (账号+密码)
    ▼
前端 (Vue3)
    │
    │ 2. Axios发送HTTP请求
    ▼
后端 FastAPI
    │
    │ 3. 验证用户凭据 → 查询PostgreSQL
    │ 4. 生成JWT Token
    │ 5. 存储用户会话到Redis
    │
    │ 6. 返回 {token, userInfo, permissions}
    ▼
前端
    │
    │ 7. 存储token到localStorage
    │ 8. 设置Axios默认header: Authorization
    │ 9. 重定向到首页
    ▼
用户浏览器显示主界面
```

### 2️⃣ 业务数据请求流程
```
前端发起请求
    │
    │ 1. GET /dev-api/system/user/list?pageNum=1
    │    Header: Authorization: Bearer {token}
    ▼
后端中间件
    │
    │ 2. JWT验证
    │ 3. 从Redis获取用户会话
    │ 4. 权限验证
    ▼
Controller层
    │
    │ 5. 参数验证
    ▼
Service层
    │
    │ 6. 业务逻辑处理
    ▼
DAO层
    │
    │ 7. SQLAlchemy查询PostgreSQL
    ▼
返回数据
    │
    │ 8. 数据序列化 (Pydantic)
    │ 9. 返回JSON响应
    ▼
前端接收
    │
    │ 10. Axios拦截器处理
    │ 11. 更新Pinia状态
    │ 12. 渲染Vue组件
```

### 3️⃣ AI推理实时流程（核心业务）
```
摄像头 (RTSP)
    │
    │ RTSP视频流
    ▼
AI推理模块 (Python)
    │
    │ 1. 视频帧解析
    │ 2. 人脸/行为检测
    │ 3. 特征提取
    │
    ├──▶ MinIO (存储图片)
    │
    ├──▶ Milvus (存储特征向量)
    │
    ├──▶ Kafka (发送识别结果)
    │        │
    │        ▼
    │    Kafka Consumer (module_kafka_bridge)
    │        │
    │        ▼
    │    处理人脸数据
    │        │
    │        ├──▶ PostgreSQL (存储告警记录)
    │        │
    │        └──▶ WebSocket (推送实时告警)
    │                  │
    │                  ▼
    │            前端实时显示
    └──▶ Redis (缓存最新状态)
```

### 4️⃣ WebSocket实时通信
```
前端 Vue3
    │
    │ 1. 建立WebSocket连接
    │    ws://localhost:9099/dev-api/websocket/{userId}
    ▼
后端 WebSocket Manager
    │
    │ 2. 维护连接池
    │ 3. 用户会话绑定
    │
    │ 当有告警产生时：
    │ 4. AI推理 → Kafka → Consumer → WebSocket Manager
    │ 5. 推送消息到指定用户/广播
    │
    │ {
    │   "type": "alarm",
    │   "data": {
    │     "cameraId": "10",
    │     "alarmType": "人脸识别",
    │     "imageUrl": "http://minio:9000/...",
    │     "time": "2025-11-06 10:30:00"
    │   }
    │ }
    ▼
前端接收
    │
    │ 6. onMessage事件触发
    │ 7. 更新告警列表
    │ 8. 弹窗提示
    │ 9. 播放提示音
```

## 📦 核心模块说明

### 后端模块结构
```
ruoyi-fastapi-backend/
├── module_admin/              # 系统管理模块（用户、角色、权限、菜单）
├── module_ai_inference/       # AI推理服务（人脸识别、行为检测）
├── module_alarm/              # 告警管理模块
├── module_camera_device/      # 摄像头设备管理
├── module_function_deployment/# 功能部署管理
├── module_streaming/          # 流媒体处理
├── module_face_search/        # 人脸搜索（Milvus）
├── module_kafka_bridge/       # Kafka数据桥接
├── module_minio/              # MinIO对象存储
├── module_monitor/            # 系统监控
└── module_vp/                 # 视频处理管道
```

### 前端模块结构
```
ruoyi-fastapi-frontend/src/
├── views/                    # 页面视图
│   ├── system/              # 系统管理（用户、角色、菜单、部门）
│   ├── monitor/             # 系统监控（在线用户、日志、缓存）
│   ├── device/              # 设备管理
│   ├── alarm/               # 告警管理
│   ├── deployment/          # 功能部署
│   └── dashboard/           # 数据大屏
├── api/                     # API接口封装
├── components/              # 公共组件
├── store/                   # Pinia状态管理
├── router/                  # 路由配置
└── utils/                   # 工具函数
```

所有API请求统一前缀：`/dev-api`

**示例：**
- 登录：`POST /dev-api/login`
- 用户列表：`GET /dev-api/system/user/list`
- 告警列表：`GET /dev-api/alarm/list`
- WebSocket：`ws://host:9099/dev-api/websocket/{userId}`
